To access the Tkinter menu load up 'MathGameUI.py' and in the main menu, where START is shown input these buttons:
	up up down down left right left right b a


To load apps in this folder:
Make a folder and call it whatever you want.
Add a .exe file inside the folder and it MUST be called 'main.exe' (this is what will run)
Load up the Tkinter menu and if a new section appears with the name of your folder the exe will run :)
